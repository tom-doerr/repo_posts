# [KartikLabhshetwar/better-shot](https://github.com/KartikLabhshetwar/better-shot)

BetterShot is an open-source alternative to CleanShot X. It offers native macOS capture, screen recording, and editing tools while prioritizing privacy through local-first operations with no subscriptions or cloud telemetry.
