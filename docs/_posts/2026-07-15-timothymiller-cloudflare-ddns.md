# [timothymiller/cloudflare-ddns](https://github.com/timothymiller/cloudflare-ddns)

Updates Cloudflare DNS records to facilitate remote access to home networks. cloudflare-ddns is a lightweight Docker container written in Rust that automatically synchronizes your public IP address to a custom domain name without requiring a static IP address.
