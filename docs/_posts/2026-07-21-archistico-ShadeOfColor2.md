# [archistico/ShadeOfColor2](https://github.com/archistico/ShadeOfColor2)

ShadeOfColor2 converts any file into a standard-looking PNG image to bypass upload restrictions, allowing users to hide data inside the pixels before decrypting it back to the original file.
