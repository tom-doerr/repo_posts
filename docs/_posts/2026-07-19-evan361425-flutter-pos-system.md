# [evan361425/flutter-pos-system](https://github.com/evan361425/flutter-pos-system)

An open-source Flutter POS system for small restaurants that manages ingredient inventory, captures customer demographics, and prints receipts via Bluetooth. The application operates offline with local-only data storage and provides custom chart analytics.
